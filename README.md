CRUD | PHP & MySQL

Aplicación web CRUD (Create, Read, Update, Delete) desarrollada en PHP y MySQL. Permite crear, visualizar, editar y eliminar registros almacenados en una base de datos, representando un caso común de desarrollo backend.
Este proyecto forma parte de mi portafolio profesional.

Tecnologías: PHP,MySQL,HTML,CSS,XAMPP.

Funcionalidades: Crear registros,Listar datos,Editar información,Eliminar registros.

Ejecución: Proyecto ejecutado en un entorno local de desarrollo utilizando XAMPP, simulando un entorno real de trabajo backend.
